Spatial Mapping Using Time-of-Flight

This project is an integrated spatial scanning system designed to visually re-map an enclosed space using LIDAR Sensor technology. The system is comprised of a TI MCU, DC motor, ToF Sensor, and a cardboard body. The ToF sensor emits a laser of infrared light, measuring the time it takes for the pulse of light to reflect to the sensor. Using the time measurement, the sensor calculates the distance data and transmits it to the MCU using the I2C communication protocol. My design records 32 distance measurement readings approximately every 11.25째 on a circulating 360째 axis around the y-z plane. Once the data is transmitted from the MCU to a connected PC, each measurement is converted into (x,y,z) coordinates in a Python script. A model of the space is then drawn using the Open3D Python Library.




To measure the distance of objects or surfaces around the device, the VL53L1X ToF Sensor takes samples of measurements every 11.25째 of the stepper motor's full rotation. This allows the sensor to collect a total of 32 distance measurements throughout the full 360째 plane revolving around the x-axis. The distance data is then transmitted to the microcontroller using the I2C serial synchronous communication protocol.




Once the data collection process is complete, the data points must be mapped in 3D for visualization of the surrounding area. When the microcontroller's program finishes its data collection state, the Python script continues to compile the data and write it into a .xyz file. Depending on the number of scans chosen by the user, the number of coordinates in the x-direction varies. The draw_geometries() function is then used on the lineset and an interactive visualization appears which can be used for observation and/or manipulation by the user.
